iii.conference Installation
---------------------------

To install iii.conference using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``iii.conference`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        iii.conference

* Re-run buildout, e.g. with:

    $ ./bin/buildout
